Project Setup with Xampp

  1. Simply download the resulta folder and paste it under Xampp/htdocs folder.
  2. Open the link http://localhost/resulta/nfl-team-list/ to see the list in a table fetched from given api.

Search

  1. On this pagehttp://localhost/resulta/nfl-team-list/ you will see there is a search bar on the top of the page.
  2. Try to search any result by typing any text, if it matches, it will displays the rows.
  3. If the text does not match, it will show Records not found !!!!

Sorting

  1. On each table heading, you will see icon just after the heading name.
  2. Clicking on arrow icon for each column, will sort the results alphabetically or numerically accordingly.
